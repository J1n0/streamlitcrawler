# 구글 플레이 게임 리뷰 감정 분석기

# 프로젝트 배경

구글 플레이 스토어의 게임 리뷰를 수집하고, 감정 분석 모델을 통해 리뷰를 긍정, 부정, 중립으로 자동 분류하는 웹 애플리케이션을 구축하였습니다.

# 프로젝트 구조
주요 기능
Streamlit 기반 웹 인터페이스
실시간 게임 리뷰 수집
감정 분석 결과 시각화
분석된 리뷰 CSV 다운로드 제공

# 필수 패키지 requirements.txt
pip install streamlit, pandas, scikit-learn, matplotlib, joblib, google-play-scraper
